# TCP-JAVA
JAVA实现TCP服务器和客户端通信以及客户端之间进行通信的小Demo,计网socket实验

# 跳转到UDP实现项目地址：



# 使用方式

1、修改src文件夹下的text.txt文件，里面存放了登录需要的用户名和密码，按照规定自定义一下

2、启动SocketTCP_Server.java，启动服务端

3、启动两个SocketTCP_Client.java，分别输入用户名密码进行登录

4、根据规则进行客户端之间的通信功能。
